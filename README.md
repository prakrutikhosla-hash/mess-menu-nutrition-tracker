# 🍽️ Mess Menu Nutrition Tracker

A simple command-line application that helps hostel and college students track their daily food intake and monitor nutritional values like carbohydrates, protein, and calories.

## 📌 Problem It Solves

Most hostel students eat whatever is served in the mess without understanding its nutritional impact. They don't know how many calories they consume, whether they're getting enough protein, or if their diet is balanced. This tracker solves that by providing an easy way to log meals and see daily and weekly nutrition reports.

## ✨ Features

- **Log Meals** – Record breakfast, lunch, dinner, and snacks with multiple food items per meal
- **Nutrition Database** – Pre-loaded with 15+ common mess foods with their nutritional values
- **Daily Summary** – View total carbs, protein, and calories consumed for the current day
- **Weekly Report** – Analyze eating patterns over the last 7 days
- **Data Persistence** – All data saved locally in JSON format, available across program sessions
- **Simple CLI** – Easy-to-use menu system, no technical expertise required

## 🛠️ Technologies Used

- Python 3.x
- JSON (for data storage)
- Datetime module (for date handling)
- File I/O (for persistent storage)
